## Overview

This project demonstrates a simple, reproducible Python data analysis workflow using river discharge data.

The pipeline takes a raw dataset containing missing and invalid values, cleans the data, and performs basic exploratory analysis. The results include summary statistics and visualisations showing average river discharge by country and changes over time.

The project is structured to reflect a typical data workflow:

Raw data → data cleaning → processed dataset → exploratory analysis → saved figures

All analysis steps are automated through a pipeline script so the entire workflow can be reproduced with a single command.

## Project Structure

data/raw/  
Original dataset.

data/processed/  
Cleaned dataset generated by the cleaning script.

src/  
Python scripts for cleaning data, performing analysis, and running the pipeline.

reports/figures/  
Saved charts produced during exploratory data analysis.

requirements.txt  
Python dependencies required to run the project.

README.md  
Project description and instructions.

## How to Run the Project

1. Install dependencies

pip install -r requirements.txt

2. Run the full pipeline

python src/run_pipeline.py

This will:

- clean the raw dataset
- generate a processed dataset
- create visualisations in the reports/figures directory
